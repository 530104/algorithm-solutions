# 分组背包

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：动态规划

---

## 📝 题目描述

有 N 组物品和一个容量为 V 的背包。每组物品有若干个，同组内最多选一个。第 i 组第 j 个物品体积 vij，价值 wij。求最大价值。0 < N, V <= 100。

**输入**：第一行 N V。接下来 N 组，每组第一行 Si（物品数），接下来 Si 行每行 vij wij。

**输出**：最大价值。

---

## 💡 思路

`f[i][j]` = 前 i 组、体积 <= j 的最大价值。对每组：不选 → `f[i-1][j]`；选组内某个 k → `f[i-1][j-v[i][k]] + w[i][k]`。一维优化后 `j` 从大到小遍历，组内再枚举每个物品。

---

## 🔑 关键代码

```cpp
int n, m;
int v[110][110], w[110][110], f[110], s[110];

int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++) {
        scanf("%d", &s[i]);
        for (int j = 1; j <= s[i]; j++)
            scanf("%d%d", &v[i][j], &w[i][j]);   // 第i组第j个物品
    }
    for (int i = 1; i <= n; i++)                  // 枚举每组
        for (int j = m; j >= 1; j--)              // 体积从大到小（01背包思想）
            for (int k = 1; k <= s[i]; k++)       // 枚举组内物品
                if (v[i][k] <= j)
                    f[j] = max(f[j], f[j - v[i][k]] + w[i][k]);
                    // 不选 vs 选第i组第k个

    printf("%d", f[m]);
    return 0;
}
```

---

## 📝 总结

分组背包 = 01 背包的推广：每组相当于一个"超级物品"，只能选一个。三重循环顺序：组 → 体积（倒序）→ 组内物品。
