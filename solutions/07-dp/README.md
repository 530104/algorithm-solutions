# 动态规划

## 内容概览

动态规划（Dynamic Programming）：通过定义状态和转移方程，将复杂问题分解为重叠子问题。

## 常见技巧

- 01 背包 / 完全背包
- 线性 DP
- 区间 DP
- 状态设计
- 滚动数组优化
