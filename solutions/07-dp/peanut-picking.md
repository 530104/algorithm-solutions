# 摘花生

> **难度**：简单 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：动态规划

---

## 📝 题目描述

R 行 C 列的网格，从左上角出发，只能向右或向下走。每个格子有一定数量的花生，求最多能摘多少颗。T 组数据，1 <= T <= 100，R, C <= 100。

**输入**：第一行 T。每组第一行 R C，接下来 R 行每行 C 个整数。

**输出**：每组一行，摘到的最大花生数。

**输入样例**：
```
2
2 2
1 1
3 4
2 3
2 3 4
1 6 5
```

**输出样例**：
```
8
16
```

---

## 💡 思路

`f[i][j]` = 走到 (i,j) 的最大花生数。只能从左或上转移：`f[i][j] = max(f[i-1][j], f[i][j-1]) + w[i][j]`。

---

## 🔑 关键代码

```cpp
const int N = 110;
int n, m;
int w[N][N];
int f[N][N];

int main() {
    int T;
    scanf("%d", &T);
    while (T--) {
        scanf("%d%d", &n, &m);
        for (int i = 1; i <= n; i++)
            for (int j = 1; j <= m; j++)
                scanf("%d", &w[i][j]);           // 读入花生数

        for (int i = 1; i <= n; i++)
            for (int j = 1; j <= m; j++)
                f[i][j] = max(f[i-1][j], f[i][j-1]) + w[i][j];
                // 从上或左转移，加上当前格

        printf("%d\n", f[n][m]);
    }
    return 0;
}
```

---

## 📝 总结

经典的网格 DP（数字三角形变体）。`max(f[i-1][j], f[i][j-1])` 保证每一步都做出最优选择，且只需保存前一行和前一个状态。
