# 数字三角形

> **难度**：简单 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：动态规划

---

## 📝 题目描述

给定一个数字三角形，从顶部到底部有很多条不同的路径。每一步只能从当前数走到正下方或右下方。把路径上的数加起来，求最大的和。

示例：
```
        7
      3   8
    8   1   0
  2   7   4   4
4   5   2   6   5
```

**输入**：第一行 N（1 < N <= 1000），接下来 N 行数字三角形。

**输出**：最大和。

**输入样例**：
```
5
7
3 8
8 1 0
2 7 4 4
4 5 2 6 5
```

**输出样例**：
```
30
```

---

## 💡 思路

自底向上 DP：从倒数第二行开始，每个位置加上它正下方和右下方中的较大值，逐步上推。`f[i][j] += max(f[i+1][j], f[i+1][j+1])`，最终 `f[1][1]` 即最大和。

---

## 🔑 关键代码

```cpp
const int N = 510;
int n;
int f[N][N];

int main() {
    cin >> n;
    for (int i = 1; i <= n; i++)            // 读入三角形
        for (int j = 1; j <= i; j++)
            cin >> f[i][j];

    for (int i = n - 1; i; i--)             // 自底向上 DP
        for (int j = 1; j <= i; j++)
            f[i][j] += max(f[i + 1][j], f[i + 1][j + 1]);
            // 当前 += 下方或右下方的较大值

    cout << f[1][1] << endl;                // 顶部即为最大和
    return 0;
}
```

---

## 📝 总结

自底向上的倒推法避免了边界判断，比自顶向下更简洁。每个子问题只依赖下一行的两个值，天然满足 DP 无后效性。
