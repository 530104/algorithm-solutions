# 最长公共子序列

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：动态规划

---

## 📝 题目描述

给定两个长度分别为 N 和 M 的字符串 A 和 B，求既是 A 的子序列又是 B 的子序列的最长字符串长度。N, M <= 1000。

**输入**：第一行 N M，第二行字符串 A，第三行字符串 B（小写字母）。

**输出**：最长公共子序列长度。

**输入样例**：
```
4 5
acbd
abedc
```

**输出样例**：
```
3
```

---

## 💡 思路

`f[i][j]` = A 的前 i 个字符与 B 的前 j 个字符的 LCS 长度。若 `a[i] == b[j]`：`f[i][j] = f[i-1][j-1] + 1`；否则 `f[i][j] = max(f[i-1][j], f[i][j-1])`。

---

## 🔑 关键代码

```cpp
const int N = 1010;
int n, m;
char a[N], b[N];
int f[N][N];                              // f[i][j] = a前i个与b前j个的LCS长度

int main() {
    cin >> n >> m >> a + 1 >> b + 1;      // 从下标1开始读入

    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++) {
            if (a[i] == b[j])              // 当前字符相同
                f[i][j] = f[i-1][j-1] + 1; // 都选，长度+1
            else                           // 当前字符不同
                f[i][j] = max(f[i-1][j], f[i][j-1]); // 至少舍弃一个
        }

    cout << f[n][m] << endl;
    return 0;
}
```

---

## 📝 总结

LCS 是双串 DP 的典型状态设计。理解两种转移：(1) 字符相同时必然都选；(2) 不同时至少有一个不在 LCS 中，取最大值。
