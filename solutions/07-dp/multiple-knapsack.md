# 多重背包

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：动态规划

---

## 📝 题目描述

有 N 种物品和一个容量为 V 的背包。第 i 种物品最多有 si 件，每件体积 vi，价值 wi。求总体积不超过 V 时的最大价值。0 < N <= 1000，0 < V <= 2000，si <= 2000。

**输入**：第一行 N V，接下来 N 行每行 vi wi si。

**输出**：最大价值。

**输入样例**：
```
4 5
1 2 3
2 4 1
3 4 3
4 5 2
```

**输出样例**：
```
10
```

---

## 💡 思路

**朴素做法**：三重循环，直接枚举每种物品选 k 件。**二进制优化**：将 si 拆成 1, 2, 4, 8, ... 的若干组，每组视为一个独立的 01 背包物品，O(N*V*log S)。

---

## 🔑 关键代码

```cpp
// 二进制优化：将每种物品按 2^k 拆包，转化为 01 背包
const int N = 12010, M = 2010;
int n, m;
int v[N], w[N];    // 拆分后的物品数组
int f[M];

int main() {
    cin >> n >> m;
    int cnt = 0;                    // 拆分后物品总数
    for (int i = 1; i <= n; i++) {
        int a, b, s;
        cin >> a >> b >> s;         // 体积、价值、数量
        int k = 1;                  // 从 1 开始拆
        while (k <= s) {
            cnt++;
            v[cnt] = a * k;         // 整组体积
            w[cnt] = b * k;         // 整组价值
            s -= k;                 // 减去已拆
            k *= 2;                 // 翻倍
        }
        if (s > 0) {                // 剩余部分
            cnt++;
            v[cnt] = a * s;
            w[cnt] = b * s;
        }
    }
    n = cnt;                        // 转化为 n 个 01 背包物品

    for (int i = 1; i <= n; i++)    // 01 背包
        for (int j = m; j >= v[i]; j--)
            f[j] = max(f[j], f[j - v[i]] + w[i]);

    cout << f[m] << endl;
    return 0;
}
```

---

## 📝 总结

二进制优化将多重背包的 O(V*sum(si)) 降为 O(V*sum(log si))。核心思想：任何整数都可以用二进制拆分成 log 级别的组，每组可选或不选，等价于原问题的 0~si 件选择。
