# 01 背包

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：动态规划

---

## 📝 题目描述

有 N 件物品和一个容量为 V 的背包。每件物品只能使用一次。第 i 件体积 vi，价值 wi。求总体积不超过 V 时的最大价值。0 < N, V <= 1000。

**输入**：第一行 N V，接下来 N 行每行 vi wi。

**输出**：最大价值。

**输入样例**：
```
4 5
1 2
2 4
3 4
4 5
```

**输出样例**：
```
8
```

---

## 💡 思路

`f[i][j]` 表示前 i 件物品、体积不超过 j 的最大价值。对第 i 件：不选则 `f[i-1][j]`；选则 `f[i-1][j-vi] + wi`。取两者最大值。一维优化：`j` 从大到小遍历，`f[j] = max(f[j], f[j-v[i]] + w[i])`。

---

## 🔑 关键代码

```cpp
const int N = 1007;
int n, m;
int v[N], w[N];
int f[N];                                // 一维优化：f[j] = 体积 <= j 的最大价值

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) cin >> v[i] >> w[i];

    for (int i = 1; i <= n; i++)         // 枚举每件物品
        for (int j = m; j >= v[i]; j--)  // 体积从大到小（保证每件只用一次）
            f[j] = max(f[j], f[j - v[i]] + w[i]);
            // 不选第i件 vs 选第i件

    cout << f[m] << endl;
    return 0;
}
```

---

## 📝 总结

01 背包是所有背包问题的基础。一维优化中 `j` 从大到小的原因是保证 `f[j-v[i]]` 来自上一轮（即不含第 i 件的状态），防止一件物品被重复使用。
