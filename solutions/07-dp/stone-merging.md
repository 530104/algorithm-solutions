# 石子合并（区间 DP）

> **难度**：困难 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：动态规划

---

## 📝 题目描述

N 堆石子排成一排，每次合并相邻两堆，代价为两堆质量之和，合并后新堆与相邻石子相邻。求总代价的最小值。N <= 300。

**输入**：第一行 N，第二行 N 个整数（每堆质量 <= 1000）。

**输出**：最小总代价。

**输入样例**：
```
4
1 3 5 2
```

**输出样例**：
```
22
```

---

## 💡 思路

区间 DP：`f[i][j]` = 合并区间 [i, j] 的最小代价。枚举最后一次合并的分界点 k，`f[i][j] = min(f[i][k] + f[k+1][j]) + sum[i..j]`。按区间长度从小到大递推。

---

## 🔑 关键代码

```cpp
const int N = 307;
int a[N], s[N];                    // a: 石子质量, s: 前缀和
int f[N][N];                       // f[i][j] = 合并[i,j]的最小代价

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
        s[i] = s[i - 1] + a[i];    // 前缀和，快速求区间和
    }
    memset(f, 0x3f, sizeof f);     // 初始化为无穷大

    for (int len = 1; len <= n; len++)              // 枚举区间长度
        for (int i = 1; i + len - 1 <= n; i++) {    // 枚举左端点
            int j = i + len - 1;                     // 右端点自动确定
            if (len == 1) { f[i][j] = 0; continue; } // 单堆无需合并

            for (int k = i; k < j; k++)              // 枚举分界点
                f[i][j] = min(f[i][j],
                    f[i][k] + f[k+1][j] + s[j] - s[i-1]);
                    // 左半代价 + 右半代价 + 本次合并代价
        }

    cout << f[1][n] << endl;                         // 合并整个区间的答案
    return 0;
}
```

---

## 📝 总结

区间 DP 的通用模板：`len + 左端点 + 分界点` 三重循环。关键是理解 `f[i][j] = min(f[i][k] + f[k+1][j]) + cost(i,j)`，最后合并的代价始终是区间总和。
