# 完全背包

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：动态规划

---

## 📝 题目描述

有 N 种物品和一个容量为 V 的背包，每种物品无限件可用。第 i 种体积 vi，价值 wi。求总体积不超过 V 时的最大价值。0 < N, V <= 1000。

**输入**：第一行 N V，接下来 N 行每行 vi wi。

**输出**：最大价值。

**输入样例**：
```
7 884
6148 6786
8500 8620
6136 6930
8700 6732
409 3388
9925 642
4324 5208
```

**输出样例**：
```
6776
```

---

## 💡 思路

与 01 背包的唯一区别：每件物品可选无限次。`f[i][j] = max(f[i-1][j], f[i][j-vi] + wi)`。一维优化：`j` 从小到大遍历即可，因为 `f[j-vi]` 来自当前轮（允许重复选）。

---

## 🔑 关键代码

```cpp
const int N = 1010;
int n, m;
int v[N], w[N];
int f[N];                                // f[j] = 体积 <= j 的最大价值

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) cin >> v[i] >> w[i];

    for (int i = 1; i <= n; i++)         // 枚举每种物品
        for (int j = v[i]; j <= m; j++)  // 体积从小到大（允许重复选）
            f[j] = max(f[j], f[j - v[i]] + w[i]);

    cout << f[m] << endl;
    return 0;
}
```

---

## 📝 总结

完全背包与 01 背包代码几乎一样，唯一的区别是 `j` 的遍历方向：01 背包从大到小（每件一次），完全背包从小到大（每件无限次）。这也是区分两种背包模型的关键记忆点。
