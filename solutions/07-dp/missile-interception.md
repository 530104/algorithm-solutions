# 拦截导弹

> **难度**：困难 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：动态规划

---

## 📝 题目描述

导弹拦截系统第一发炮弹能到达任意高度，之后每发不能高于前一发。输入导弹依次飞来的高度（<= 30000 的正整数）。求：(1) 最多能拦截多少导弹；(2) 拦截所有导弹最少需要几套系统。

**输入**：一行，导弹高度序列。

**输出**：两行：最多拦截数、最少系统数。

**输入样例**：
```
389 207 155 300 299 170 158 65
```

**输出样例**：
```
6
2
```

---

## 💡 思路

**第一问**：求最长不上升子序列（LNDS），`f[i]` 以 h[i] 结尾、`h[j] >= h[i]`。

**第二问**：等价于求最少的不上升子序列覆盖数，由 Dilworth 定理，等于最长上升子序列（LIS）的长度。贪心维护多套系统的当前最低高度。

---

## 🔑 关键代码

```cpp
const int N = 1010;
int n, h[N], f[N], q[N];

int main() {
    string line;
    getline(cin, line);
    stringstream ssin(line);
    while (ssin >> h[n]) n++;

    int res = 0, cnt = 0;                     // res: 最多拦截, cnt: 最少系统
    for (int i = 0; i < n; i++) {
        f[i] = 1;
        for (int j = 0; j < i; j++)
            if (h[i] <= h[j])                 // 不上升：后面不能高于前面
                f[i] = max(f[i], f[j] + 1);
        res = max(res, f[i]);

        int k = 0;
        while (k < cnt && q[k] < h[i]) k++;   // 贪心：找第一套能拦截的系统
        if (k == cnt) q[cnt++] = h[i];        // 现有系统都拦不了，新增一套
        else q[k] = h[i];                     // 更新该系统的最低高度
    }
    printf("%d\n%d\n", res, cnt);
    return 0;
}
```

---

## 📝 总结

两问分别对应两种 LIS 变体：最长不上升子序列 + 最少覆盖数。后者等于最长上升子序列（Dilworth 定理），贪心维护多个候选系统的当前最低高度即可。
