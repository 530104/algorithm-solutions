# 归并排序

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：分治算法

---

## 📝 题目描述

给定长度为 n 的整数数列，使用归并排序从小到大排序并输出。1 <= n <= 100000。

**输入**：第一行 n，第二行 n 个整数。

**输出**：一行排好序的 n 个整数。

**输入样例**：
```
5
3 1 2 4 5
```

**输出样例**：
```
1 2 3 4 5
```

---

## 💡 思路

归并排序：将区间一分为二，递归排序两个子区间，然后用双指针将两个有序子数组合并为一个有序数组。分治到底（区间长度为 1 时自然有序），再逐层合并返回。

---

## 🔑 关键代码

```cpp
int tmp[N];

void merge_sort(int q[], int l, int r) {
    if (l >= r) return;                      // 递归边界
    int mid = (l + r) / 2;
    merge_sort(q, l, mid);                   // 递归排序左半
    merge_sort(q, mid + 1, r);               // 递归排序右半

    int k = 0, i = l, j = mid + 1;
    while (i <= mid && j <= r)               // 双指针合并两个有序数组
        if (q[i] < q[j]) tmp[k++] = q[i++];
        else             tmp[k++] = q[j++];
    while (i <= mid) tmp[k++] = q[i++];      // 剩余元素
    while (j <= r)   tmp[k++] = q[j++];
    for (i = l, j = 0; i <= r; i++, j++)     // 拷贝回原数组
        q[i] = tmp[j];
}
```

---

## 📝 总结

归并排序与快排对比：归并是先递归后合并（自底向上），快排是先分区后递归（自顶向下）。归并排序天然稳定，在求逆序对等问题中不可或缺。
