# 快速选择第 k 个数

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：分治算法

---

## 📝 题目描述

给定长度为 n 的整数数列，用快速选择算法求第 k 小的数。1 <= n <= 100000，1 <= k <= n。

**输入**：第一行 n 和 k，第二行 n 个整数。

**输出**：第 k 小的数。

**输入样例**：
```
5 3
2 4 1 5 3
```

**输出样例**：
```
3
```

---

## 💡 思路

快速选择算法是快排的变体：每次分区后，判断 k 落在左半还是右半区间，只递归搜索含 k 的那一半，另一半直接舍弃，平均 O(n)。

（本题代码用了先全排序再取 k，也可通过。）

---

## 🔑 关键代码

```cpp
void quick_sort(int q[], int l, int r) {
    if (l >= r) return;
    int i = l - 1, j = r + 1, x = q[l + r >> 1];
    while (i < j) {
        do i++; while (q[i] < x);
        do j--; while (q[j] > x);
        if (i < j) swap(q[i], q[j]);
    }
    quick_sort(q, l, j);       // 递归左半
    quick_sort(q, j + 1, r);   // 递归右半
}
// 排序后，直接取第 k-1 个位置（或只递归含 k 的区间）
```

---

## 📝 总结

快排框架稍作修改即可变为快速选择：只需要递归搜索第 k 个元素所在的那一半，时间复杂度优化到 O(n)。
