# 分治算法

## 内容概览

分治（Divide and Conquer）：将问题拆成多个子问题，各自解决后合并结果。

## 常见技巧

- 二分分治（归并排序、快速排序）
- 分治 + 递归树分析
- 主定理求时间复杂度
