# 输出前 k 大的数

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：分治算法

---

## 📝 题目描述

给定数组，统计前 k 大的数并从大到小输出。n < 100000，k < n。

**输入**：第一行 n，第二行 n 个整数，第三行 k。

**输出**：前 k 大的数，每行一个，从大到小。

**输入样例**：
```
10
4 5 6 9 8 7 1 2 3 0
5
```

**输出样例**：
```
9
8
7
6
5
```

---

## 💡 思路

对数组排序后，从末尾向前取 k 个即可得到前 k 大的数（降序输出）。排序可用快速排序（分治）或归并排序，时间复杂度 O(n log n)。

---

## 🔑 关键代码

```cpp
quick_sort(q, 0, n - 1);            // 快速排序（升序）
int cnt = 0;
for (int i = n - 1; cnt < k; i--) { // 从后往前取 k 个
    printf("%d\n", q[i]);
    cnt++;
}
```

---

## 📝 总结

排序后从大到小取数是最直观的做法。若 n 极大而 k 很小，可用堆或快速选择优化到 O(n log k)。
