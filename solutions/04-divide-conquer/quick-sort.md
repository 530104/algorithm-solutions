# 大数排序

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：分治算法

---

## 📝 题目描述

给定长度为 n 的整数数列（1 <= n <= 100000），所有整数在 1~10^9 范围内，将数列从小到大排序后输出。

**输入**：第一行 n，第二行 n 个整数。

**输出**：一行排好序的 n 个整数。

**输入样例**：
```
12
111584322 907287398 13562672 615771042 83035480 323016588 501254568 158361642 388135644 58329696 400904710 38908140
```

**输出样例**：
```
13562672 38908140 58329696 83035480 111584322 158361642 323016588 388135644 400904710 501254568 615771042 907287398
```

---

## 💡 思路

快速排序（Quick Sort）：选一个基准值 x，将数组分成左右两部分——左边所有数小于 x，右边所有数大于 x。然后递归对左右两部分分别排序，规模不断缩小直到区间长度为 1。

---

## 🔑 关键代码

```cpp
void quick_sort(int q[], int l, int r) {
    if (l >= r) return;                          // 区间只剩1个或0个元素，无需排序
    int i = l - 1, j = r + 1, x = q[l + r >> 1]; // 双指针初始化为边界外，取中间值为基准
    while (i < j) {
        do i++; while (q[i] < x);                // 左指针右移，找到第一个 >= x 的数
        do j--; while (q[j] > x);                // 右指针左移，找到第一个 <= x 的数
        if (i < j) swap(q[i], q[j]);             // 交换，使小数去左边、大数去右边
    }
    quick_sort(q, l, j);                         // 递归排序左半部分
    quick_sort(q, j + 1, r);                     // 递归排序右半部分
}
```

---

## 📝 总结

快速排序是分治思想的经典应用。每次选基准值将区间一分为二，左右递归排序，平均时间复杂度 O(n log n)。注意双指针的初始位置和基准值的选取影响边界处理。
