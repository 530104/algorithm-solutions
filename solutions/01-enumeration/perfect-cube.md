# 完美立方

> **难度**：简单 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：枚举算法

---

## 📝 题目描述

形如 `a^3 = b^3 + c^3 + d^3` 的等式被称为完美立方等式。编写一个程序，对任给的正整数 `N  (N <= 100)`，寻找所有的四元组 `(a, b, c, d)`，使得 `a^3 = b^3 + c^3 + d^3`，其中 `a, b, c, d` 大于 1，小于等于 `N`，且 `b <= c <= d`。

## 输入

一个正整数 `N  (N <= 100)`。

## 输出

每行输出一个完美立方，格式为：`Cube = a, Trip<= = (b,c,d)`。按 `a` 从小到大输出，`a` 相同时 `b` 小的优先，依次类推。

**输入样例**：
```
24
```

**输出样例**：
```
Cube = 6, Trip<= = (3,4,5)
Cube = 12, Trip<= = (6,8,10)
Cube = 18, Trip<= = (2,12,16)
Cube = 18, Trip<= = (9,12,15)
Cube = 19, Trip<= = (3,10,18)
Cube = 20, Trip<= = (7,14,17)
Cube = 24, Trip<= = (12,16,20)
```

---

## 💡 思路

枚举 `b`、`c`、`d` 三个正整数，计算它们的立方和，判断是否等于某个整数 `a` 的立方。

---

## 🔑 关键代码

```cpp
for (int a = 2; a <= N; ++a)                // a 范围 [2, N]
    for (int b = 2; b <= a - 1; ++b)        // b 范围 [2, a-1]
        for (int c = b; c <= a - 1; ++c)    // c 范围 [b, a-1]
            for (int d = c; d <= a - 1; ++d) // d 范围 [c, a-1]
                if (a*a*a == b*b*b + c*c*c + d*d*d)
                    printf("Cube = %d, Trip<= = (%d,%d,%d) n", a, b, c, d);
```

---

## 📝 总结

利用数学关系 `b, c, d < a` 和 `b <= c <= d` 缩小枚举范围，按从小到大的顺序自然输出结果，无需额外排序。
