# 枚举算法

## 内容概览

枚举（Enumeration）是最基础的算法思想：列出所有可能的情况，逐一检查，找出符合条件的解。

## 常见技巧

- 多重循环枚举
- 剪枝：提前排除不可能的分支
- 枚举顺序优化
