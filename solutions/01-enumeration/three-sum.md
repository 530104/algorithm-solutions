# 三数之和

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：枚举算法

---

## 📝 题目描述

给定一个目标值 `target`，在整数数组 `a` 中找出三个元素 `(x, y, z)` 使 `x + y + z == target`。找出所有满足条件的三元组，按从小到大顺序输出，三元组中要求 `x < y < z` 且不允许重复数字。

**输入**：第一行有两个整数 `target` 和 `n`。第二行为数组 `a` 的 `n` 个元素，用空格隔开。

**输出**：所有满足条件的三元组，按 `x` 升序，`x` 相同按 `y` 升序。

**输入样例**：
```
17 7
0 2 7 10 15 18 25
```

**输出样例**：
```
0 2 15
0 7 10
```

---

## 💡 思路

先将数组排序并去重，固定第一个数 `nums[i]` 后，在右侧区间 `[i+1, n-1]` 内用双指针 `j`（左）和 `k`（右）枚举，两数之和偏大时 `k` 左移缩小和，找到等值则记录结果。

---

## 🔑 关键代码

```cpp
sort(nums.begin(), nums.end());                         // 升序排序
nums.erase(unique(nums.begin(), nums.end()), nums.end()); // 去重

for (int i = 0; i < nums.size(); i++) {                 // 固定第一个数
    for (int j = i + 1, k = nums.size() - 1; j < k; j++) {  // j 左指针，k 右指针
        while (j < k - 1 && nums[i] + nums[j] + nums[k - 1] >= target) k--;  // 右指针左移
        if (nums[i] + nums[j] + nums[k] == target)      // 找到三元组
            res.push_back({nums[i], nums[j], nums[k]});
    }
}
```

---

## 📝 总结

排序 + 去重 + 双指针将三重枚举优化到 $O(n^2)$，关键在于内层 `k` 指针随 `j` 增加单调左移，避免了不必要的检查。
