# 两数之和

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：枚举算法

---

## 📝 题目描述

给定一个目标值 `target`，在不包含重复元素的按升序排列的整数数组 `a` 中，找出和为目标值的那两个整数，并返回他们的数组下标。

**输入**：第一行有两个整数 `target` 和 `n`，分别代表目标值和数组元素个数。第二行为数组 `a` 的 `n` 个元素，每个元素用空格隔开。

**输出**：输出和为 `target` 的两个元素的下标 `i j`（其中 `i < j`）。

**输入样例**：
```
17 7
1 3 5 7 10 11 19
```

**输出样例**：
```
3 4
```

---

## 💡 思路

枚举所有数对 `(i, j)`，其中 `i < j`，逐一检查 `nums[i] + nums[j]` 是否等于目标值 `target`，找到即输出下标。

---

## 🔑 关键代码

```cpp
for (int j = 0; j < nums.size(); j++) {        // 枚举第二个数下标
    for (int i = 0; i < j; i++) {              // 枚举第一个数下标（保证 i < j）
        if (nums[i] + nums[j] == target) {     // 判断两数之和是否等于目标值
            cout << i << " " << j << endl;     // 输出结果
            break;
        }
    }
}
```

---

## 📝 总结

最直观的暴力枚举思路，固定 `j` 后向前枚举 `i`，`i < j` 的条件既避免了重复也保证了输出顺序。
