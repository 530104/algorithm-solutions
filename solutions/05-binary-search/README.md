# 二分算法

## 内容概览

二分查找（Binary Search）：在有序区间内不断缩小搜索范围，每次都排除一半。

## 常见技巧

- 标准二分查找
- 二分答案
- 边界处理（左闭右闭 vs 左闭右开）
- 浮点数二分
