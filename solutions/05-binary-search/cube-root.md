# 数的三次方根

> **难度**：简单 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：二分算法

---

## 📝 题目描述

给定浮点数 n（-10000 <= n <= 10000），求它的三次方根，保留 6 位小数。

**输入样例**：
```
1000.00
```

**输出样例**：
```
10.000000
```

---

## 💡 思路

二分答案：初始区间 [-10000, 10000]，中点 mid 的立方与 n 比较。若 `mid^3 >= n`，答案在左侧 `r = mid`；否则 `l = mid`。浮点数二分直至区间足够小。

---

## 🔑 关键代码

```cpp
double n;
cin >> n;
double l = -10000, r = 10000;        // n 范围 [-10000, 10000]，三次方根不会超出此范围
while (r - l >= 1e-8) {              // 保留6位小数，精度取1e-8（多两位）
    double mid = (l + r) / 2;
    if (mid * mid * mid >= n)        // mid的立方 >= 目标值
        r = mid;                     // 答案在左侧（或等于mid）
    else
        l = mid;                     // 答案在右侧
}
printf("%.6lf", l);
```

---

## 📝 总结

整数二分和浮点数二分的区别：浮点没有 `+1`/`-1` 的边界问题，直接赋值 `l = mid` 或 `r = mid`。注意当 n 为负数时三次方根也为负，区间 `[-10000, 10000]` 可覆盖。
