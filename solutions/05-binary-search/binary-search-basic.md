# 查找指定数

> **难度**：简单 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：二分算法

---

## 📝 题目描述

在 N 个不重复的升序数组 nums 中查找 target，返回下标，找不到输出 -1。N <= 100000。

**输入**：第一行 N，第二行 N 个整数，第三行 T（查询次数），接下来 T 行每行一个 target。

**输出**：T 行，每行为下标或 -1。

**输入样例**：
```
36
0 1 5 8 10 12 15 17 20 26 36 68 71 80 90 92 96 100 101 104 130 275 345 405 425 519 573 583 608 616 714 780 802 842 910 961
5
8
42
64
130
912
```

**输出样例**：
```
3
-1
-1
20
-1
```

---

## 💡 思路

二分查找：每次取区间中点 mid，若 `p[mid] >= target`，目标在左侧（含 mid）；否则目标在右侧。最后检查 `p[l]` 是否等于 target。

---

## 🔑 关键代码

```cpp
int bsearch(int l, int r, int target) {
    while (l < r) {
        int mid = l + r >> 1;      // 取区间中点
        if (p[mid] >= target)      // 中点值 >= 目标，答案在左半（含mid）
            r = mid;
        else                       // 中点值 < 目标，答案在右半
            l = mid + 1;
    }
    // 最终 l == r，区间缩为一个点
    if (p[l] == target) return l;  // 找到
    else return -1;                // 不存在
}
```

---

## 📝 总结

二分查找模板一（`r = mid` 分支）：适用于找第一个 >= target 的位置，`mid = l + r >> 1` 下取整配合 `l = mid + 1` 保证不陷入死循环。
