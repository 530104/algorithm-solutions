# 突袭路线（拓扑排序）

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：广度优先搜索（BFS）

---

## 📝 题目描述

给定 n 个点 m 条边的有向图，求任意一个拓扑序列（若存在）。若不存在输出 -1。拓扑序列：对于每条边 (x, y)，x 都出现在 y 之前。n, m <= 10^5。

**输入**：第一行 n m，接下来 m 行每行 x y。

**输出**：拓扑序列（一行 n 个数）或 -1。

**输入样例**：
```
3 3
1 2
2 3
1 3
```

**输出样例**：
```
1 2 3
```

---

## 💡 思路

Kahn 算法（BFS 思想）：维护每个点的入度 `d[i]`。先将所有入度为 0 的点入队。每次从队头取出一个点 x 加入结果序列，遍历 x 的所有出边，将邻居入度减 1，若减为 0 则入队。若最终序列长度 == n，说明存在拓扑序；否则图中存在环，输出 -1。

---

## 🔑 关键代码

```cpp
const int N = 10000007;
int n, m;
int head[N], edge[N], nextvertex[N], idx;     // 邻接表
int d[N];                                      // 入度数组
int q[N];                                      // 数组模拟队列 + 结果序列

bool topsort() {
    int head = 0, tail = -1;
    for (int i = 1; i <= n; i++)               // 所有入度为0的点入队
        if (d[i] == 0)
            q[++tail] = i;

    while (head <= tail) {
        int x = q[head++];                     // 取出队头
        for (int i = h[x]; i != -1; i = nextvertex[i]) {
            int y = edge[i];
            d[y]--;                            // "删除" x→y，y 入度减1
            if (d[y] == 0)                     // y 入度变0，可以入队
                q[++tail] = y;
        }
    }
    return tail == n - 1;                      // 队列包含了所有点 → 拓扑序存在
}
// 若有解，q[0..n-1] 就是拓扑序列
```

---

## 📝 总结

拓扑排序是 DAG 判定的经典算法。核心思想：每次删除入度为 0 的点及所有出边，类似 BFS 的层序遍历。若最终不能删除所有点，说明存在环，无法排出拓扑序。
