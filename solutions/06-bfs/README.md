# 广度优先搜索（BFS）

## 内容概览

BFS（Breadth-First Search）：逐层扩展，先近后远，适合求最短路径。

## 常见技巧

- 队列实现 BFS
- 层序遍历
- 最短路径问题
- 多源 BFS
- 双向 BFS
