# 净化迷雾森林（广搜）

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：广度优先搜索（BFS）

---

## 📝 题目描述

森林是一个 n x m 的矩阵，`.` 表示空地，`#` 表示障碍，`@` 表示起点。每次可向上下左右移动一格，问总共能净化多少个格子（含起点）。

**输入**：多组数据，每组第一行 m n（行列），接下来 n 行每行 m 个字符。以 `0 0` 结束。

**输出**：每组输出净化格数。

---

## 💡 思路

BFS 漫水填充：从起点入队，逐层向四个方向扩展，每访问一个 `.` 标记为 `#` 防重复，计数器 +1。队列空时所有可达区域统计完成。

---

## 🔑 关键代码

```cpp
const int N = 25;
char g[N][N];
int n, m;
int dx[] = {-1, 0, 1, 0}, dy[] = {0, 1, 0, -1};  // 上右下左

int bfs(int sx, int sy) {
    queue<PII> q;
    q.push({sx, sy});             // 起点入队
    g[sx][sy] = '#';              // 标记已访问，防止重复
    int res = 0;

    while (q.size()) {            // 队列非空继续搜索
        auto t = q.front();       // 取队头
        q.pop();                  // 弹出队头
        res++;                    // 计数+1

        for (int i = 0; i < 4; i++) {           // 搜索四个方向
            int x = t.first + dx[i], y = t.second + dy[i];
            if (x < 0 || x >= n || y < 0 || y >= m || g[x][y] != '.') continue;
            g[x][y] = '#';         // 标记已访问
            q.push({x, y});        // 新坐标入队
        }
    }
    return res;
}
```

---

## 📝 总结

BFS 与 DFS 漫水填充效果相同，但 BFS 用队列而非递归，天然适合求最短距离；DFS 用递归栈，适合回溯计数。
