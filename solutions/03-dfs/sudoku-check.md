# 记忆碎片验证

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：深度优先搜索（DFS）

---

## 📝 题目描述

给定初始数独矩阵 A（9x9，空白处用 0 表示），输入一个填满的 9x9 矩阵，判断它是否是初始矩阵 A 的有效解。需满足：

1. 输入数字必须与初始矩阵中非 0 位置完全一致
2. 每行、每列、每个 3x3 宫内均为 1-9 各出现一次

初始矩阵 A：
```
530070000
600195000
098000060
800060003
400803001
700020006
060000280
000419005
000080079
```

**输出**：若是有效解输出 `Yes`，否则 `No`。

---

## 💡 思路

分四步验证：先检查输入格式是否合法（9 行 9 列，每个字符为数字），再验证是否与初始矩阵兼容（非 0 位置必须匹配），最后分别检查每行、每列、每个 3x3 宫是否有重复数字。

---

## 🔑 关键代码

```cpp
bool row() {
    for (int i = 0; i < 9; i++) {
        bool st[10] = {false};                 // 标记当前行已出现的数字
        for (int j = 0; j < 9; j++) {
            int t = b[i][j];
            if (t < 1 || t > 9 || st[t])       // 数字无效或重复
                return false;
            st[t] = true;
        }
    }
    return true;
}
// col() 同理，遍历每列
// block() 遍历9个3x3宫，起点 (x,y) 步长3
```

---

## 📝 总结

数独验证模板题。行、列、宫的检查各自独立，用布尔数组标记出现过的数字即可，不需要复杂数据结构。
