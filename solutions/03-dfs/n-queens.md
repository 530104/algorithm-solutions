# N 皇后

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：深度优先搜索（DFS）

---

## 📝 题目描述

输入一个正整数 `N`，输出 `N` 皇后问题的全部摆法。在 `N  times N` 的棋盘上放置 `N` 个皇后，使其互不攻击（任意两个皇后不在同一行、同一列、同一对角线）。

输出结果里每一行代表一种摆法，行里第 `i` 个数字如果是 `k`，代表第 `i` 行的皇后放在第 `k` 列。行列编号从 1 开始。

**输入**：皇后的个数 `n`（n <= 13）

**输出**：每行一种摆法，输出长度为 `n` 的正整数。

**输入样例**：
```
4
```

**输出样例**：
```
2413
3142
```

---

## 💡 思路

用 DFS 按行递归放置皇后，每一行枚举所有列，检查当前列和对角线是否已被占用。若合法则放置并递归到下一行，放置完 `N` 行后输出一种摆法，然后回溯尝试其他列。

---

## 🔑 关键代码

```cpp
void dfs(int row) {
    if (row > n) {                   // 所有行都放置完毕
        for (int i = 1; i <= n; i++) // 输出当前摆法
            cout << pos[i];
        cout << endl;
        return;
    }
    for (int col = 1; col <= n; col++) {         // 枚举每一列
        if (!colUsed[col] &&                      // 列未被占用
            !diag1[row + col] &&                   // 主对角线未被占用
            !diag2[row - col + n]) {               // 副对角线未被占用
            pos[row] = col;                        // 第row行皇后放在col列
            colUsed[col] = diag1[row + col] = diag2[row - col + n] = true;
            dfs(row + 1);                          // 递归下一行
            colUsed[col] = diag1[row + col] = diag2[row - col + n] = false; // 回溯
        }
    }
}
```

---

## 📝 总结

DFS 回溯的经典应用。用三个布尔数组标记列和两条对角线的占用状态，将冲突检查从 `O(n)` 降为 `O(1)`，是本题的核心优化。
