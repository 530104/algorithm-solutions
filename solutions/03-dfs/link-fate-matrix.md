# 林克的命运之阵

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：深度优先搜索（DFS）

---

## 📝 题目描述

有一个无穷大的方格矩阵，从第 1 行某个格子出发，只能向下、左、右三个方向走，走过的格子不重复。问走 `n` 步共有多少种不同的路径方案？（2 种走法只要有一步不一样即视为不同）

**输入**：步数 `n`（`n  <= 20`）

**输出**：不同路径的总数。

**输入样例**：
```
2
```

**输出样例**：
```
7
```

---

## 💡 思路

从起点开始 DFS，每步可以选择下、左、右三个方向之一。用标记数组记录走过的格子避免重复访问，步数用完时计数 +1。回溯时清除标记，继续尝试其他方向。

---

## 🔑 关键代码

```cpp
int dx[3] = {0, 0, 1};           // 左、右、下
int dy[3] = {-1, 1, 0};

long long dfs(int i, int j, int n) {
    if (n == 0) return 1;        // 步数用完，计数+1
    long long res = 0;
    fates[i][j] = 1;             // 标记已走过
    for (int k = 0; k < 3; k++) {
        int x = i + dx[k], y = j + dy[k];
        if (fates[x][y] == 0)    // 未走过才能走
            res += dfs(x, y, n - 1);
    }
    fates[i][j] = 0;             // 回溯：清除标记
    return res;
}
```

---

## 📝 总结

典型网格 DFS + 回溯计数。起点取矩阵中间避免出界。`n=20` 时结果已达千万级，无需记忆化也能在可接受时间内跑完。
