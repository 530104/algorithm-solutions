# 寻找林克的回忆(1)

> **难度**：困难 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：深度优先搜索（DFS）

---

## 📝 题目描述

把 9x9 的数独补充完整，使得每行、每列、每个 3x3 的九宫格内数字 1~9 均恰好出现一次。数字为 0 表示空格。若有多解输出任意一种。

**输入**：9 行，每行 9 个数字（0 代表空格）。

**输出**：9 行填满的数独。

**输入样例**：
```
103000509
002109400
000704000
300502006
060000050
700803004
000401000
009205800
804000107
```

**输出样例**：
```
143628579
572139468
986754231
391542786
468917352
725863914
237481695
619275843
854396127
```

---

## 💡 思路

逐行逐列 DFS：每个位置若已有数字则跳过；若为空，枚举 1-9，用三个布尔数组分别记录每行、每列、每个 3x3 宫已使用的数字。填入数字后递归下一格，填完 9 行找到解直接返回。

---

## 🔑 关键代码

```cpp
const int N = 9;
int g[N][N];
bool st_row[N][N + 1], st_col[N][N + 1], st_block[3][3][N + 1];

bool dfs(int x, int y) {
    if (y == 9) x++, y = 0;                     // 当前行搜完，进入下一行
    if (x == 9) {                                // 搜完所有行，输出解
        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) cout << g[i][j];
            cout << endl;
        }
        return true;
    }
    if (g[x][y] != 0) return dfs(x, y + 1);     // 已有数字，跳过

    for (int t = 1; t <= 9; t++) {               // 枚举 1-9
        if (!st_row[x][t] && !st_col[y][t] && !st_block[x / 3][y / 3][t]) {
            st_row[x][t] = st_col[y][t] = st_block[x / 3][y / 3][t] = true;
            g[x][y] = t;
            if (dfs(x, y + 1)) return true;      // 找到解立即返回
            g[x][y] = 0;                         // 回溯
            st_row[x][t] = st_col[y][t] = st_block[x / 3][y / 3][t] = false;
        }
    }
    return false;                                // 无解
}
```

---

## 📝 总结

DFS 解决数独的基本框架：三个布尔数组做冲突检查，回溯搜索所有空格。逐格逐行扫描保证了搜索的完整性。
