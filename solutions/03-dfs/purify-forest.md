# 净化迷雾森林

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：深度优先搜索（DFS）

---

## 📝 题目描述

给定一个 `W x H` 的矩阵，`.` 代表可净化区域，`#` 代表墙壁，`@` 代表起点。从起点出发，只能上下左右移动到相邻的 `.` 区域，求能净化区域的总数（包含起点）。

**输入**：多个数据集合，每集合第一行是 `W` 和 `H`（W, H <= 20），接下来 `H` 行每行 `W` 个字符。以 `0 0` 结束。

**输出**：每集合一行，输出可净化区域数量。

**输入样例**：
```
6 9
....#.
.....#
......
......
......
......
......
#@...#
.#..#.
0 0
```

**输出样例**：
```
45
```

---

## 💡 思路

DFS 漫水填充（Flood Fill）：从 `@` 出发，递归向四个方向搜索，每遇到一个 `.` 就计数并标记为 `#`（已访问），不再重复走。

---

## 🔑 关键代码

```cpp
int dx[4] = {-1, 0, 1, 0};       // 上、右、下、左
int dy[4] = {0, 1, 0, -1};

int dfs(int i, int j) {
    int res = 1;                  // 当前格子算1个
    forest[i][j] = '#';           // 标记已访问，防止重复
    for (int k = 0; k < 4; k++) {
        int x = i + dx[k], y = j + dy[k];
        if (x >= 0 && x < h && y >= 0 && y < w && forest[x][y] == '.')
            res += dfs(x, y);     // 累加四个方向的连通区域
    }
    return res;
}
```

---

## 📝 总结

DFS 漫水填充模板题。核心是四个方向遍历 + 标记已访问，将走过的 `.` 改为 `#` 省去了布尔标记数组。
