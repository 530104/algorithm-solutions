# 深度优先搜索（DFS）

## 内容概览

DFS（Depth-First Search）：沿着一条路径走到底，再回溯探索其他分支。

## 常见技巧

- 递归实现 DFS
- 回溯（Backtracking）
- 排列、组合、子集问题
- 棋盘类问题
