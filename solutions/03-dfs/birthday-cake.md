# 蛋糕塔

> **难度**：困难 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：深度优先搜索（DFS）

---

## 📝 题目描述

制作一个 `N` 层圆柱体蛋糕，总体积为 `V \cdot \pi`。从底层往顶层数，第 `i` 层半径为 `r_i`，高度为 `h_i`，满足 `r_i > r_{i+1}` 且 `h_i > h_{i+1}`（`r_i`、`h_i` 均为整数）。求使得表面积 `Q = S \cdot \pi` 最小的 `S`。`V \<= 100000`，`N \<= 20`。

**输入**：第一行 `V`，第二行 `N`。

**输出**：最小 `S`，无解输出 0。

**输入样例**：
```
100
2
```

**输出样例**：
```
68
```

---

## 💡 思路

DFS + 强剪枝：从底层向上逐层确定半径和高度，用三种剪枝策略缩小搜索空间——体积超额剪枝、面积不比当前最优解优剪枝、以及利用剩余体积估算最小侧面积剪枝。

---

## 🔑 关键代码

```cpp
void dfs(int u, int v, int s) {
    if (v + minv[u] > n) return;              // 剪枝1：剩余最小体积超出
    if (s + mins[u] >= ans) return;            // 剪枝2：当前面积已不小于最优解
    if (s + 2 * (n - v) / R[u + 1] >= ans) return; // 剪枝3：侧面积预判

    if (!u) {                                  // 所有层处理完毕
        if (v == n) ans = s;
        return;
    }
    for (int r = min(R[u+1]-1, (int)sqrt(n-v)); r >= u; r--)      // 半径从大到小
        for (int h = min(H[u+1]-1, (n-v)/r); h >= u; h--) {        // 高度从大到小
            int t = 0;
            if (u == m) t = r * r;                                  // 最底层加顶面积
            R[u] = r; H[u] = h;
            dfs(u - 1, v + r*r*h, s + 2*r*h + t);
        }
}
```

---

## 📝 总结

DFS 求解最优解时，剪枝是关键。本题的三种剪枝——体积下界、面积下界、侧面积放缩估计，将搜索空间从指数级缩减到可接受范围。
