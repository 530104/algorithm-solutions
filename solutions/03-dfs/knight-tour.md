# 骑士的怜悯 (1)

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：深度优先搜索（DFS）

---

## 📝 题目描述

骑士（马）在国际象棋棋盘上按 8 个方向移动。给定变形的 `p x q` 棋盘（1 <= p * q <= 26），判断骑士能否从某个格子出发，不重复地遍历所有格子，输出字典序最小的路径。若不可行则输出 `none`。

**输出格式**：每个格子用大写字母（列）加数字（行）表示，如 `A1B3C1...`；路径的字典序由字母顺序决定。

**输入样例**：
```
5
5 1
5 2
5 3
5 4
5 5
```

**输出样例**：
```
#1:
none
#2:
none
#3:
none
#4:
A1B3A5C4D2B1A3B5D4C2B4A2C1D3C5A4B2D1C3D5
#5:
A1B3A5C4A3B1D2E4C5A4B2D1C3B5D4E2C1A2B4D5E3C2E1D3E5
```

---

## 💡 思路

DFS 回溯：从每个格子出发尝试遍历整棋盘。定义 8 个移动方向的顺序使其天然满足字典序（列字母小的优先），找到第一条完整路径即最优，直接输出并返回。

---

## 🔑 关键代码

```cpp
int dx[8] = {-2, -2, -1, -1, 1, 1, 2, 2};   // 移动方向按字母顺序排列
int dy[8] = {-1, 1, -2, 2, -2, 2, -1, 1};

bool dfs(int x, int y, int cnt) {
    path.push_back({(char)(x + 'A'), y + 1}); // 记录当前格子坐标
    if (cnt == p * q) return true;            // 所有格子遍历完毕
    st[x][y] = true;
    for (int i = 0; i < 8; i++) {
        int a = x + dx[i], b = y + dy[i];
        if (a < 0 || a >= q || b < 0 || b >= p || st[a][b]) continue;
        if (dfs(a, b, cnt + 1)) return true; // 找到即返回，保证字典序最小
    }
    st[x][y] = false;                         // 回溯
    path.pop_back();
    return false;
}
```

---

## 📝 总结

骑士巡游是 DFS 回溯的经典题。方向数组顺序决定了路径的字典序，`return true` 保证了找到第一条合法路径即为最优解。
