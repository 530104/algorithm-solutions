# 寻找林克的回忆(2)

> **难度**：困难 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：深度优先搜索（DFS）

---

## 📝 题目描述

把 9x9 的数独补充完整，使得每行、每列、每个 3x3 的九宫格内数字 1~9 均恰好出现一次。与 (1) 不同的是本题多组输入，以 `end` 结束。每组输入为一行 81 字符的字符串，`.` 代表空格。

**输入**：多组数据，每组一行 81 字符。以 `end` 结束。

**输出**：每组数据输出一行填满的 81 字符。

**输入样例**：
```
4.....8.5.3..........7......2.....6.....8.4......1.......6.3.7.5..2.....1.4......
......52..8.4......3...9...5.1...6..2..7........3.....6...1..........7.4.......3.
end
```

**输出样例**：
```
417369825632158947958724316825437169791586432346912758289643571573291684164875293
```

---

## 💡 思路

位运算优化 + 启发式剪枝：用 9 位二进制数表示每行/列/宫可用数字（1 对应 0 位）。每次选择**可选数字最少**的空格来填，大幅减少搜索分支。`lowbit` 快速枚举可选项，`ones` 表统计可选数个数。

---

## 🔑 关键代码

```cpp
int ones[1 << N], LOG[1 << N];           // ones: 二进制中1的个数, LOG: lowbit对应的位数
int row[N], col[N], block[3][3];         // 每行/列/宫的可用数字状态（二进制位为1表示可用）

inline int get(int x, int y) {
    return row[x] & col[y] & block[x / 3][y / 3];  // 三集合取交集
}

bool dfs(int cnt) {
    if (!cnt) return true;               // 全部空格填完

    int minv = 10, x, y;
    for (int i = 0; i < N; i++)          // 找可选数最少的空格（启发式剪枝）
        for (int j = 0; j < N; j++)
            if (str[i * 9 + j] == '.') {
                int t = ones[get(i, j)];
                if (t < minv) { minv = t; x = i; y = j; }
            }

    for (int i = get(x, y); i; i -= lowbit(i)) {  // 枚举每个可用数字
        int t = LOG[lowbit(i)];
        row[x] -= 1 << t;                // 将第t位置0（标记已使用）
        col[y] -= 1 << t;
        block[x / 3][y / 3] -= 1 << t;
        str[x * 9 + y] = '1' + t;

        if (dfs(cnt - 1)) return true;   // 递归下一层

        row[x] += 1 << t;                // 回溯
        col[y] += 1 << t;
        block[x / 3][y / 3] += 1 << t;
        str[x * 9 + y] = '.';
    }
    return false;
}
```

---

## 📝 总结

与 (1) 的逐格尝试不同，本题两大优化：(1) 位运算替代布尔数组，用 `&` 快速求可用数交集；(2) 每次选分支最少的空格，将搜索树宽度降至最小，是数独 DFS 的标准高效写法。
