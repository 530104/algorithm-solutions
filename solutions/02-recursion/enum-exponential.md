# 递归实现指数型枚举

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：递归算法

---

## 📝 题目描述

从 $1 \sim n$ 这 $n$ 个整数中随机选取任意多个，输出所有可能的选择方案。

**数据范围**：$1 \le n \le 15$

**输出**：每行一种方案，同一行内的数升序排列，相邻两个数用一个空格隔开。没有选任何数的方案输出空行。各行之间顺序任意。

**输入样例**：
```
3
```

**输出样例**：
```

3
2
2 3
1
1 3
1 2
1 2 3
```

（第一行空行表示不选任何数）

---

## 💡 思路

对每个数 $k$ 有两种选择：选或不选。递归处理 $k$ 到 $n$，到达 $n+1$ 时输出当前选中的集合。

---

## 🔑 关键代码

```cpp
void dfs(int k) {
    if (k > n) {                    // 递归边界：n个数都决定完了
        for (int i = 1; i <= n; i++)
            if (chosen[i]) cout << i << " ";
        cout << endl;
        return;
    }
    chosen[k] = false;              // 不选第k个数
    dfs(k + 1);
    chosen[k] = true;               // 选第k个数
    dfs(k + 1);
}
```

---

## 📝 总结

指数型枚举是最基础的递归枚举，每个元素独立做"选/不选"的决策，共 $2^n$ 种方案。
