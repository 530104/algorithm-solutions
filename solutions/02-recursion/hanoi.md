# 汉诺塔 I

> **难度**：简单 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：递归算法

---

## 📝 题目描述

有三根杆子 A、B、C。A 杆上有 $N$ 个穿孔圆盘，尺寸由下到上依次变小。要求按下列规则将所有圆盘移至 C 杆：

- 每次只能移动一个圆盘
- 大盘不能叠在小盘上面
- 圆盘可临时置于 B 杆

问：最少需要多少步？输出每一步的移动。

**输入**：盘子个数 `n`（1 <= n <= 20）

**输出**：每行一步移动，格式 `A->C`。

**输入样例**：
```
3
```

**输出样例**：
```
A->C
A->B
C->B
A->C
B->A
B->C
A->C
```

---

## 💡 思路

把 $n$ 个盘从 A 移到 C，分解为三步：先把 $n-1$ 个盘从 A 移到 B（借助 C），再把第 $n$ 个盘从 A 移到 C，最后把 $n-1$ 个盘从 B 移到 C（借助 A）。规模缩小为 $n-1$，递归求解。

---

## 🔑 关键代码

```cpp
void hanoi(int n, char a, char b, char c) {
    if (n == 1) {                    // 只剩一个盘，直接移动
        cout << a << "->" << c << endl;
        return;
    }
    hanoi(n - 1, a, c, b);          // n-1个从A移到B（借助C）
    cout << a << "->" << c << endl; // 第n个从A移到C
    hanoi(n - 1, b, a, c);          // n-1个从B移到C（借助A）
}
```

---

## 📝 总结

递归的核心是把大问题转化为同样结构的小问题。汉诺塔最小移动次数为 $2^n - 1$，是递归思想的经典范例。
