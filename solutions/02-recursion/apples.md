# 放苹果

> **难度**：简单 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：递归算法

---

## 📝 题目描述

有 $M$ 个同样的苹果放在 $N$ 个同样的盘子里，允许有的盘子空着不放，问共有多少种不同的放法？盘子相同、苹果相同，顺序无关（5,1,1 和 1,5,1 是同一种）。

**输入**：第一行是测试数据数目 $t$（0 <= t <= 20），以下每行两个整数 $M$ 和 $N$（0 <= M, N <= 10）。

**输出**：每组数据输出一行 $K$，代表放法的种数。

**输入样例**：
```
1
7 3
```

**输出样例**：
```
8
```

---

## 💡 思路

设 `f(m, n)` 为 $m$ 个苹果放 $n$ 个盘子的放法数。分类讨论：苹果比盘子少时，多余盘子可忽略，`f(m,n) = f(m,m)`；苹果足够时，要么至少空一个盘子 `f(m, n-1)`，要么所有盘子至少放一个 `f(m-n, n)`。递归边界：苹果数为 0 或 1 个盘子时只有 1 种放法。

---

## 🔑 关键代码

```cpp
int f(int m, int n) {
    if (m == 0 || n == 1) return 1;   // 0个苹果或只剩1个盘子，只有1种放法
    if (m < n) return f(m, m);        // 苹果少于盘子，多出的盘子忽略
    return f(m, n - 1) + f(m - n, n); // 空一个盘子 + 每个盘子至少放一个
}
```

---

## 📝 总结

递归分类讨论的经典题。关键是把原问题拆分成互斥的两类：是否有空盘子，保证不重不漏。
