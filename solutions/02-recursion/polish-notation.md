# 波兰表达式

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：递归算法

---

## 📝 题目描述

波兰表达式（前缀表达式）将运算符写在前面，如 `* + 2 3 4` 等价于中缀 `(2 + 3) * 4`。编写程序求解波兰表达式的值。

运算符包括 `+ - * /`，运算数是浮点数，运算符和运算数之间用空格分隔。

**输入样例**：
```
* + 11.0 12.0 + 24.0 35.0
```

**输出样例**：
```
1357.000000
```

---

## 💡 思路

波兰表达式天然适合递归求解：每次读取一个字符串。若为运算符，递归计算其左右两个子表达式的值并运算；若为数字，直接转换为浮点数返回。

---

## 🔑 关键代码

```cpp
double p() {
    string str;
    cin >> str;                         // 读入下一个符号
    switch (str[0]) {
        case '+': return p() + p();     // 运算符：递归求两个操作数
        case '-': return p() - p();
        case '*': return p() * p();
        case '/': return p() / p();
        default:  return stof(str);     // 数字：转为浮点数返回
    }
}
```

---

## 📝 总结

前缀表达式免去了括号和优先级判断，递归读入一个符号即可自然拆分左右子树。逆波兰表达式同理，只是读取顺序和递归方向不同。
