# 递归实现组合型枚举

> **难度**：中等 &nbsp;&nbsp;|&nbsp;&nbsp; **分类**：递归算法

---

## 📝 题目描述

从 `1  sim n` 这 `n` 个整数中随机选出 `m` 个，输出所有可能的选择方案。

**数据范围**：`n > 0`，`0 <= m <= n`，`n + (n - m) <= 25`

**输出**：按字典序从小到大输出所有方案，同一行内的数升序排列。例如 `1 3 5 7` 排在 `1 3 6 8` 前面。

**输入样例**：
```
5 3
```

**输出样例**：
```
1 2 3
1 2 4
1 2 5
1 3 4
1 3 5
1 4 5
2 3 4
2 3 5
2 4 5
3 4 5
```

---

## 💡 思路

从 `1` 开始递归，每次选择一个数后只能选更大的数，保证升序和字典序。当选满 `m` 个时输出，或者剩余可选数不够 `m` 时剪枝提前返回。

---

## 🔑 关键代码

```cpp
void dfs(int k, int start) {
    if (k > m) {                    // 已选够m个
        for (int i = 1; i <= m; i++) cout << chosen[i] << " ";
        cout << endl;
        return;
    }
    for (int i = start; i <= n; i++) {  // 从start开始往后选，保证升序
        chosen[k] = i;
        dfs(k + 1, i + 1);              // 下一个从i+1开始
    }
}
```

---

## 📝 总结

组合枚举通过"后一个数大于前一个数"保证不重复不遗漏，`start` 参数的递增保证了字典序输出。
